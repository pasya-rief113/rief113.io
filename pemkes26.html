Rekap & Statistik

Panduan Penggunaan dengan OneDrive

Langkah Impor Data dari OneDrive

  1. Buka link OneDrive: https://1drv.ms/x/c/76465615c90d6a30/IQAAzp8xhGAiSLxAzsQDxDW6ARmvPObF3Kp6YpEtOlRlwjI?e=BstTef
  2. Klik "Open in Desktop App" atau "Download" untuk menyimpan file Excel ke komputer
  3. Kembali ke aplikasi ini, klik tombol "Impor" (hijau) di header
  4. Pilih file Excel yang sudah di-download — data dari sheet "Data" akan otomatis dimuat
  5. No RM duplikat dari file akan dilewati otomatis dengan peringatan

Langkah Ekspor & Upload ke OneDrive

  1. Setelah selesai mengisi/edit data, klik tombol "Ekspor" di header
  2. File Excel akan ter-download dengan 3 sheet: "Data", "Laporan Fakultas", "Rekap RM"
  3. Buka kembali OneDrive, klik "Upload" untuk mengganti file lama dengan versi terbaru
  4. Tim lain bisa download file terbaru dan mengimpor ke aplikasi mereka
Workgroup Sync: Jika beberapa orang membuka aplikasi ini di tab browser yang sama pada 1 komputer, data tersinkronisasi otomatis secara real-time. Untuk kolaborasi antar komputer berbeda, gunakan alur Impor/Ekspor melalui OneDrive di atas.

Langkah Cetak Surat Keterangan

  1. Di tabel data, klik ikon printer pada baris pasien yang ingin dicetak suratnya
  2. Atau klik tombol "Cetak Surat" di header, lalu pilih data pasien
  3. Pilih dokter: dr. Bayu atau dr. Arfi
  4. Isi Nomor Surat dan Hasil Pemeriksaan
  5. Klik "Cetak Sekarang" — surat akan tercetak sesuai format yang ada di sheet OneDrive

Validasi No RM

  • Saat mengetik No RM, sistem langsung mengecek apakah nomor sudah dipakai
  • Jika duplikat, muncul notifikasi merah dan field berwarna merah
  • Klik tombol refresh di samping No RM untuk generate nomor otomatis
  • Format No RM: 6 digit angka (contoh: 000001, 000002, dst.)
